AI Agent API Documentation

AIエージェントを活用した新事業創出支援システムのAPIドキュメントです。リサーチからアイデア生成、評価、分析、レポート作成まで一連のワークフローを自動化できます。

API概要
基本的な使用方法と設定について

ベースURL

Development: http://localhost:3000/api
Staging: https://staging.example.com/api
Production: https://api.example.com/api

認証

すべてのAPIリクエストにはBearerトークンによる認証が必要です。

Authorization: Bearer {your_token}
Content-Type: application/json

レート制限

  • 開発環境: 無制限
  • 本番環境: 1000 req/hour per IP
  • 認証済み: 10000 req/hour per user
POST /api/agents/execute
全エージェントを順次実行して完全なビジネス分析レポートを生成

このエンドポイントは、Researcher → Ideator → Critic → Analyst → Writer の順番で全エージェントを実行し、包括的なビジネス分析レポートを生成します。

主な機能

  • 市場調査とトレンド分析
  • ビジネスアイデアの生成
  • アイデアの評価と選定
  • 詳細な市場分析
  • 構造化されたレポート生成

リクエストボディ

{
  "projectId": "uuid",
  "topic": "スマートシティ市場調査",
  "config": {
    "depth": "comprehensive",
    "timeout": 90000,
    "streaming": true
  }
}

パラメータ

  • projectId (string, optional): プロジェクトID
  • topic (string, required): 調査するトピック
  • config.depth (string): 分析の深度 (standard, comprehensive)
  • config.timeout (number): タイムアウト時間（ミリ秒）
  • config.streaming (boolean): ストリーミングレスポンスの有効化

JavaScript

const response = await fetch('/api/agents/execute', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${token}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    topic: 'スマートシティ市場調査',
    config: { depth: 'standard' }
  })
});

const data = await response.json();

Python

import requests

response = requests.post(
    'http://localhost:3000/api/agents/execute',
    headers={
        'Authorization': f'Bearer {token}',
        'Content-Type': 'application/json'
    },
    json={
        'topic': 'スマートシティ市場調査',
        'config': {'depth': 'standard'}
    }
)

data = response.json()
